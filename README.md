# go-brick

所谓 ”砖连砖成墙，瓦连瓦成房“；

go-brick 是一套Go服务基础框架，意于集成主流Go开源组件库，使它们紧密关联在一起，让搬砖更轻松。



# 功能

- 模块化，松耦合设计
- 组件丰富，简单易用
- 组件接口设计，提供中间件，易于拓展
- 详细追踪信息和错误堆栈
- 链式超时控制
- 多种泛型数据结构
- 更多功能待补充...



# 限制

```shell
golang version >= 1.19
```
